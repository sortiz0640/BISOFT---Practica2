ACERCA DE EL PROGRAMA - COMO USAR

>> Para iniciar sesion, usar los siguientes datos quemados:
    usuario: admin
    contrasena: admin

>> No hay base de datos, se maneja mediante un arreglo de js totalmente quemado 
de forma local, por lo que los datos no se guardan de forma permanente.

>> Luego de registrar un usuario, se debe iniciar sesion para ir al dashboard.

>> Si se intenta acceder al dashboard sin estar logueado, se redireccionara al login.

>> Iniciar la ejecucion con npm run dev 
>> Direccion: localhost:3000
